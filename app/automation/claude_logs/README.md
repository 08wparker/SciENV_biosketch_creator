# Claude Browser Extension Automation Logs

This folder contains JavaScript files generated by Claude in Chrome browser extension while inspecting the SciENcv website. These files document:

1. **CSS Selectors** for form elements
2. **Data structures** for each section
3. **Step-by-step automation logic**

## Files

| File | SciENcv Section |
|------|-----------------|
| `section_b_appointments_positions.js` | B. Appointments and Positions |
| `section_a_education.js` | A. Professional Preparation (Education) |
| `section_c_products.js` | C. Products |
| `supplement_a_personal_statement.js` | Supplement A. Personal Statement |
| `supplement_b_honors.js` | Supplement B. Honors |
| `supplement_c_contributions.js` | Supplement C. Contributions to Science |
| `create_document.js` | Create New Document dialog |

## Usage

These logs help develop the Playwright automation in `app/automation/sciencv_filler.py` by providing:

- Accurate CSS selectors discovered through browser inspection
- Field order and required fields
- Example data structures
- Suggested Playwright commands

## Generating New Logs

1. Open SciENcv in Chrome with Claude browser extension
2. Navigate to the section you want to document
3. Ask Claude to inspect the form and generate automation code
4. Save the output to a new `.js` file in this folder
